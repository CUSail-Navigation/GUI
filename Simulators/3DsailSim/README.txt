README FOR SAILSIM


INTRODUCTION
------------

		SailSim is a 3D dynamic simulation of a sailboat.



RUNNING THE SIMULATION
----------------------

		* Navigate to the SailSim folder

		* Type 'main' in the command window to run the main.m file

		* Hit ENTER to start the simulation



MODIFYING THE PARAMETERS OF THE BOAT/SIMULATION
-----------------------------------------------

		* Open the file 'setBoatParam.m'

		* Find and change the desired parameter

		* Save 'setBoatParam.m'
